Commands to Remember

  • man $COMMAND manual
  • ls $DIR list directory
  • touch $FILE affect a file
  • mkdir $PATH make a directory
  • rmdir $PATH remove a directory
  • cd $PATH change directory
  • pwd print working directory

More Commands to Remember

  • less $FILE display contents of file
  • cp $SRC $DEST copy a file somewhere
  • mv $SRC $DEST move/rename a file
  • rm $FILE permanently delete a file
  • sudo $CMD super user do
