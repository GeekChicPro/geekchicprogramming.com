Hello, World

Write a program that prints out a greeting to the command line.

Executing a Python Program

In order to execute a python program, you'll have open a terminal window and navigate to the directory containing your python program that you wrote in a text editor. If that program is named "hello.py" then use the following command:


 $ python hello.py

A Convention

A convention that python programmers use for programs that execute from the command line like we're doing is to put all their execution code in blocks like the following:


 if __name__ == "__main__":
     print "Good Program!"

Don't worry if you don't get it-- but it is very good to do! You'll probably thank me later!

Countdown

Write a program that prints all digits from 10 to 4

Storing State

One good thing to consider is rather than writing 6 lines of code for each print statement, is to do a little bit of math. You'll need a variable to store the number you're currently on, then update it using subtraction after you print the number out.

Keep Going!

You can't subtract only once in this case, you'll have to keep doing it until a condition is met. What will keep the program subtracting? You're probably going to have to use a for loop or a while loop to keep the subtraction going.

Knowing when to Stop

You'll have to test to make sure when to stop, otherwise you're program will just keep going forever! In this case, you need to ensure that your variable holding the current number (or state) is tested to make sure it's not less than four. If it is, then you're done!

Alternatives

There are many ways to write this program, here are a few:

  • Iterate through a list that holds the numbers 10 to 4
  • Create a while-loop that keeps subtracting until it's less than 4
  • Make a function that simply prints one less, and is called only if the number > 4

There are usually many ways to solve programming problems, how did you solve this one?

A Birthday Book

Write a program that creates a list of contacts with their name and birthday

  • Create a function that adds contacts to the list.
  • Add a few names and birthdays to your list.
  • Create another function that prints out the names and birthdays.
  • Better yet: store contacts by Birthday month, print by month.
  • Bonus: create a function that figures out the youngest person.

Data Storage

You're going to need a place to keep all of your data, and for the time being, the best way to keep it is in a list or a dictionary at the top of your program. All your functions will be able to access that list by name.

Function Parameters

In this case, your function needs to take in two parameters, name and birthday. Interestingly, you'll have to decide how you'd like to store the Birthday; should you pass the day, month, year as seperate things? Are they numbers or strings? The arguments to the function can be parameters or keyword arguments.

Nested Data Structure

You can always create seperate data structures for each month to store the birthdays by month. However, a nested data structure, e.g. a list of lists or a dictionary of lists, could help keep things more compact.

External Modules

One possible solution to the finding the youngest person is to use the built in datetime module that is in Python. Datetime allows you to do math on dates and has a particularly good and efficient data store for dates and times. Ask your instructor if you're interested. However-- there are other ways, don't be afraid to try!
