Hello, Django

The web framework for perfectionists with deadlines

  • Created to manage news sites for The World Company
  • Named after guitarist Django Reinhardt
  • Inteded to help journalists meet fast deadlines

A Pulitzer!

  • Politifact: fact checking in 2008 US Presidential Election
  • Lead developer was a former journalist
  • His first Django application
  • http://www.mattwaite.com/posts/2007/aug/22/announcing-politifact/

and more...

Principles

  • DRY (Don't repeat yourself)
  • Write less code
  • Make CRUD easy
  • Database Independent (Oracle, MySQL, PostgreSQL, SQLite)
  • Deployment neutral (Apache, Nginx)

Features

  • Object-relational mapping (ORM)
  • Build In Admin interface
  • URL Design and Control
  • Hierarchical Template System
  • il8n and l10n for localization

Model-Template-View (MVC, sorta)

Models: a thing that contains data (e.g. a blog post)

Views: figure out what to show and process

Templates: how things are displayed
