Time to Success: 30-45 minutes

In order to successfully complete this workshop, you'll have to create a development environment by installing some software and configuring your machine. This can often be difficult becuase of the various operating systems that you and your classmates have. We'll do our best to create a homogenous environment in this workshop.

Follow the instructions for your operating system. If you run into any problems, don't panic- invite the instructor over!

  • Setup for OS X
  • Setup for Windows
  • Setup for Ubuntu

Setup for OS X

Step 1. Install Python

GREAT NEWS! Since you're on OS X, Python has already been installed for you. Most Unix/Linux systems come with Python already on it!

To verify, open a terminal (see Step 2) and type python at the prompt. you should see something very similar to the code below:

        $ python
        Python 2.7.2 (default, Oct 11 2012, 20:14:37) 
        [GCC 4.2.1 Compatible Apple Clang 4.0 (tags/Apple/clang-418.0.60)] on darwin
        Type "help", "copyright", "credits" or "license" for more information.
        >>>

Type exit() to get out of python interpreter.

Step 2. Install a Terminal

MORE GOOD NEWS! Again, OS X already comes with a built in Terminal that's ready to use. You can find it in the Utilities directory of your Applications folder. You can also type in Terminal in Spotlight search and that should open it up also. It might be a good idea to add this to your dock, or create a shortcut on your desktop, since we're going to be using it a fair amount.

  1. Go to your Applications directory
  2. Double click Utilities
  3. Drag the Terminal to your Dock

If you're familiar with the Terminal, ask your instructor about iTerm - a Mac Terminal with more functionality.

Step 3. Install Django

Now the fun begins!

  1. Download the latest release of Django
  2. Go to your Downloads directory
  3. Double click the Django-1.5.1.tar.gz file -- it should unpack to a folder
  4. Open a Terminal (see Step 2)
  5. Type: cd ~/Downloads/Django-1.5.1
  6. Type: sudo python setup.py install
  7. You'll probably have to type the password to your computer now.
  8. You should see a bunch of stuff start scrolling down the screen
  9. Check to see if it worked:
    1. In the terminal type python
    2. You should see the >>> prompt.
    3. Type: import django
    4. Nothing should happen If you get an error, tell an instructor.
    5. Type: exit()
  10. Done!

Step 4. Install Sublime Text

Most people are surprised to find out that all you need to code is a simple text editor- nothing fancy. In fact, you want the simplest possible editor you can find! Although there are many Integrated Development Environments (IDEs) out there, we'll use Sublime Text:

  1. Download Sublime Text and install it

Step 5. Setup the Project Files

We've pre-written some code for you to make the short time we have in the workshop go easier.

  1. Download the Project
  2. Unpack the zip folder on your desktop

Setup for Windows

Step 1. Install Python

Unfortunately Windows doesn't natively come with Python so we're going to have to do some wrangling to get it on your system.

  1. Download version 2.7 of Python
  2. Use the Windows installer to install Python

You now have Python installed on your computer. You may want to look into editing the PATH environment variable to ensure that python is available in any Command Prompt. However, we will not be using native Windows command prompts in this class.

You can test Python by opening up an IDLE terminal from your applications or Start menu.

Step 2. Install a Terminal

So that everyone is on the same page, for this class, we're going to use Cygwin as our emulator. In the work environment, it is likely that you'll have to use a Linux or Unix Terminal, so it's good to get used to it.

  1. Download setup.exe from Cygwin
  2. Right click setup.exe and Run as Administrator
  3. Follow the Instructions in the setup suite
  4. When you get to the section where it prompts you for software:
    1. Go to the "interpreters" section
    2. Ensure that python is checked to Installed

At this point you might want to create a shortcut to the cygwin application on your desktop or in your quick launch bar since we'll be using it frequently.

Step 3. Install Django

Installing Django is also going to be a bit tricky.

  1. Download the latest release of Django
  2. Open a Cygwin Terminal as Administrator (see Step 2)
  3. Type: cd ~/Downloads/
  4. Type: tar -xzvf Django-1.5.1.tar.gz -- it should unpack to a folder
  5. Type: cd Django-1.5.1
  6. Type: python setup.py install
  7. You should see a bunch of stuff start scrolling down the screen
  8. Check to see if it worked:
    1. In the terminal type python
    2. You should see the >>> prompt.
    3. Type: import django
    4. Nothing should happen If you get an error, tell an instructor.
    5. Type: exit()
  9. Done!

Step 2. Install a Terminal

MORE GOOD NEWS! Again, Ubuntu already comes with a built in Terminal that's ready to use. Since we'll be using this often, you'll want to create a launcher on your desktop or in the side dock.

If you're using Unity:

  1. Open the Dash
  2. Type Terminal and hit Return
  3. Keyboard Shortcut: CTRL + ALT + T

If you're using the old style menu:

  1. Go to Applications → Accessories → Terminal
  2. Keyboard Shortcut: CTRL + ALT + T

Create a Launcher:

  1. Right-click any unused space in any panel
  2. Choose Add to Panel...
  3. Fill in Name (Terminal)
  4. Use Browse to select the Terminal
  5. Click OK
