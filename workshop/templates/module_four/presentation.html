Markup

Structure that both Humans and Computers can interpret

Three key components to Markup:

  • Tags
  • Attributes
  • Elements

Tags

Opening and Closing Tags surround values.


 <tag>element</tag>

Nesting: Tags can surround other tags.


 <tag><inner>element</inner></tag>

Attributes

Extra information associated with tags. These are key/value pairs that inside the opening tag. The value must be surrounded by quotation marks.


 <tag attribute="value">element</tag>

Elements

Tags mark the beginning and end of elements. Elements are your content, and can either be text or another nested structure of tags.

An HTML Page


 <!DOCTYPE html>

 </head>
